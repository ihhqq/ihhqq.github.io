Windows版v2rayN使用教程

2023-08-05 PV:

简介

V2rayN 是一款运行在 Windows 平台的 v2ray 客户端，基于 v2fly_core 和 xray_core， 支持 VMess、VLESS、Shadowsocks、Socket、Trojan 多种类型。支持订阅，添加节点的方法也比较多样，可以手动添加，从剪切板导入，扫描屏幕二维码。

下面来做一个简单的入门操作步骤。

一、下载启动

点这里下载 V2rayN

V2ray-Core 下载完成后是一个 zip 包，解压缩，然后打开目录里面的 v2rayN.exe应用程序。

二、导入节点

导入节点的方法有三种

1.手动添加

手动添加就不介绍了，就是把参数一个个的写入到配置里面，这个不推荐，容易出错，也慢，尤其是节点比较多的时候。

2.扫描屏幕二维码

顾名思义，就是在你电脑上打开一个二维码图片时在你的客户端中依次点 服务器 -> 扫描屏幕上的二维码 就可以导入节点了。

点这里查看免费节点（每日更新）

3.从粘贴板导入当你已经复制好节点链接后 [vmess, vless, trojan, ss]， 在 v2rayN 客户端中依次点 服务器 -> 从剪切板批量导入URL便可以导入链接了。

三、启动节点

点击右下角的图标，系统代理 -> 自动配置系统代理

回到客户端中选择你想要使用的节点，鼠标右击并设置为活动服务器

这样就完成了 v2rayN 的客户端节点导入与设置，下面就可以科学上网了。

四、订阅

订阅的话，就是订阅一个链接，这个链接包含有批量的节点，订阅服务器上的节点有变化时，在客户端只需要更新订阅，那么客户端的节点也会跟着变化。下面说一下操作步骤

客户端中选择 订阅 -> 订阅设置

备注随便写，链接写订阅服务器的链接，点确定

回到客户端点 订阅 -> 更新订阅, 如果链接有效的话，可以看到客户端多了很多节点

五、总结

V2rayN 是一款操作简单，容易上手的 v2ray 客户端，免安装，打开导入节点直接使用，可以说没有任何门槛。跟着这个步骤来一定可以学会。如果没有节点，可以使用我站提供的免费节点，每天更新，可以流畅播放4K视频。
